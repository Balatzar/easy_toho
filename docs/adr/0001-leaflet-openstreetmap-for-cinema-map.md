# Use Leaflet and OpenStreetMap for the Cinema Map

Easy Toho eventually needs a real Tokyo map for choosing a Cinema, but version one will ship without a map so the app can answer the immediate showtime-planning need faster. When the map is added, we will use Leaflet with OpenStreetMap tiles because it gives us real geography and simple selectable pins without API-key setup, Google Maps billing, or the design burden of maintaining a custom schematic map.

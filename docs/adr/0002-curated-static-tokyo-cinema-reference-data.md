# Use Curated Static Data for Tokyo Cinemas

Easy Toho will fetch live screening data, but Tokyo Cinema names, official venue identifiers, and Cinema-level IMAX Capability labels will be curated as static app reference data. This keeps Cinema selection reliable without scraping venue metadata on every request, while still preserving the no-user-persistence constraint; addresses and map coordinates can be added to the same reference data when the v2 map ships.
